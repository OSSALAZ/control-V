# Control Vial - 

Demo de plataforma móvil para el sistema de control vehicular del municipio.

## Características

- ✅ Registro e inicio de sesión de ciudadanos
- ✅ Módulo "Mi Cuenta" con datos personales y direcciones
- ✅ Gestión de vehículos (registro, visualización, edición)
- ✅ Visualización de centros de atención
- ✅ Solicitud de tag RFID con pasarela de pago
- ✅ Opción de recoger tag en módulo o envío a domicilio
- ✅ Diseño responsive optimizado para dispositivos móviles

## Uso

1. Abre el archivo `index.html` en un navegador web
2. La aplicación está diseñada para verse como una app móvil (ancho máximo 414px)
3. Puedes iniciar sesión con cualquier email y contraseña (demo)
4. Explora todos los módulos disponibles

## Estructura

- `index.html` - Estructura principal de la aplicación
- `styles.css` - Estilos con paleta de colores del proyecto
- `app.js` - Lógica y funcionalidad de la aplicación
- `Assets/logo.png` - Logo del municipio

## Paleta de Colores

- Azul oscuro (#002E5F) - Marca/cabecera
- Azul medio (#005A9C) - Botones y elementos destacados
- Blanco (#FFFFFF) - Fondos principales
- Gris claro (#F4F5F6) - Fondos alternativos
- Gris/Negro (#1D1D1D / #333333) - Texto principal

## Notas

Esta es una demo funcional. Los datos se almacenan temporalmente en memoria del navegador y se perderán al recargar la página.
